Box Editor for Tesseract

Box Editor Words Pairing
Welcome!

Upload an image and a .box file. Unique words will automatically be extracted and added to a sorted words list. Pair up each word with its translation to create a mapping file.

Transliteration progress is shown below the image. When done, download the .wordlist file and the .json file to extract transliteration rules.

Tips

Select characters in the input field to see their Unicode name.

While transliterating use return and shift+return to move forward and backward.
