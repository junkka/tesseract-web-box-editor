Box Editor for Tesseract Beta

Welcome!

Upload an image and start tagging the text using the form. Bounding boxes will automatically be generated in a few seconds.

You can also upload a .box file to start from a previous state, which will discard auto-generated bounding boxes.

Keyboard Shortcuts

While tagging use return and shift+return to move forward and backward.
